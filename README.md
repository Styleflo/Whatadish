# Whatadish

Whatadish est un application web, dont le but est l'apprentissage de la culture anglaise à travers différents plats anglophones.
Le but deviner la provenance d'un plat.

## Installation

### Étape 1 : cloner le projet
```bash
  git clone 
```

### Étape 2 : créer un environement python
```bash
  python -m venv venv
  source venv/bin/activate
```

### Étape 3 : installer les dépendances
```bash
  pip install requirements.txt
```

### Étape 4 : Lancer le projet
```bash
  python app.py
```
